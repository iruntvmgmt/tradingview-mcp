# Sprint 1 ‚Äî Foundation (Phase 0 + Phase 1)

**Goal:** Scaffold the repo, build the CDP transport layer, recon tooling, error types, and DOM utilities. End state: a working `recon_findings.json` with all 24 capabilities classified.

## Files to Create

### 1. `pyproject.toml`
- Project name: `tv-desktop-controller`
- Python 3.11+
- Dependencies: `mcp`, `websockets`, `httpx` (for HTTP‚ÜíWS upgrade), `click` (for recon CLI)
- Dev dependencies: `pytest`, `pytest-asyncio`

### 2. `core/services/cdp_connection.py`
- `class CDPConnection`:
  - `launch()` ‚Äî spawn TV Desktop with `--remote-debugging-port=8315`
  - `connect()` ‚Äî WebSocket connect to `ws://127.0.0.1:8315` with retry/backoff (3 retries, 2s delay)
  - `list_targets()` ‚Äî enumerate `Target.getTargets`, find the main renderer
  - `select_main_renderer_target()` ‚Äî filter for `about:blank` or the main tradingview window
  - `execute_js(expression)` ‚Äî `Runtime.evaluate` via CDP
  - `listen_network()` ‚Äî enable `Network` domain, capture XHR/WebSocket frames
  - `disconnect()` ‚Äî clean close
  - `health_check()` ‚Äî ping via `Runtime.evaluate('1+1')`

### 3. `core/services/errors.py`
- `class TvMcpError(Exception)` ‚Äî base, with `code` and `details` fields
- `class ConnectionError(TvMcpError)` ‚Äî CDP connection failures
- `class CapabilityUnavailable(TvMcpError)` ‚Äî capability not in recon or marked unavailable
- `class BackendConfigurationError(TvMcpError)` ‚Äî unknown path type
- `class SelectorResolutionError(TvMcpError)` ‚Äî DOM selector not found
- `class OrderSubmissionBlocked(TvMcpError)` ‚Äî confirmed=False guard
- `class ReplayStateError(TvMcpError)` ‚Äî wrong replay mode state

### 4. `core/services/dom_utils.py`
- `class DomUtils`:
  - `resolve_selector(selectors: list[str])` ‚Äî try fallback selectors in order, return first match
  - `click(selectors)` ‚Äî resolve + CDP `Input.dispatchMouseEvent`
  - `type_text(selectors, text)` ‚Äî resolve + CDP `Input.insertText`
  - `extract_table(selectors, row_selectors)` ‚Äî resolve + `Runtime.evaluate` to scrape DOM table
  - `wait_until(selectors, timeout=10)` ‚Äî poll until selector resolves
  - `scroll_paginated_list(container_selectors)` ‚Äî scroll event via CDP
  - `click_at_coordinates(container_selectors, x_ratio, y_ratio)` ‚Äî new method for canvas drawing placement

### 5. `core/services/recon.py`
- `class ReconRunner`:
  - `dump_window_globals()` ‚Äî `Runtime.evaluate` to list `window` keys, look for tradingview internals
  - `probe_known_paths()` ‚Äî try known JS paths (`window.tradingView`, `window.tvWidget`, etc.)
  - `tap_network()` ‚Äî enable Network domain, print guided instruction sequence (~4 min protocol)
  - `dump_dom_structure()` ‚Äî snapshot key UI panels (alert modal, drawing toolbar, order ticket, replay toolbar, Pine editor, Strategy Tester)
  - `report()` ‚Äî output structured `recon_findings.json`
- Async CLI entry point using `asyncio`

### 6. `recon_findings.json`
- Initial template with all 24 capability keys, all paths set to `"dom"`, all `verified: false`
- Generated by first recon run
